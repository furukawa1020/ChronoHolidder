# ChronoHolidder (クロノホリダー) ⛏️🕰️

**これは、あなたの足元を掘り返す「時空発掘ツール」です。**

**「その場所、いつが一番盛り上がってた？」**
今立ってるその地面。1年前かもしれないし、10年前かもしれないし、江戸時代かもしれないし、なんなら人間がいなかったカンブリア紀かもしれない。
ChronoHolidderは、観光名所を案内するのではなく、**「今いるその座標のピーク（最高潮）」** を1億年単位のシームレスな地層から特定し、物理的に掘り起こす（Visual Excavation）ためのツールです。

## 思想 (Philosophy)
*   **Dig, Don't Walk:** 「見て回る」のではなく「掘る」。画面上の土をボロボロと崩し、地層を剥がして初めて真実が見える。
*   **Seamless Time:** 「歴史（数百年）」と「地質（数億年）」を区別しない。1年前の流行も、白亜紀の恐竜も、等しく「過去のレイヤー」として扱う。
*   **Thermal Peak:** その土地が最も「熱かった（Hot）」瞬間を特定する。有名な寺がなくても、恐竜が大量に死んだ場所ならそこがピーク。

## 機能 (Features)
*   **Seamless Scoring Engine**: Wiki(歴史)とGBIF(化石)のデータを混ぜ合わせ、現代から太古までを等列に並べて「密度の濃い時代」を計算。
*   **Excavation UI (AR/Touch)**: ただ表示するだけではつまらない。画面を覆う「現代の地層」を指で削り（Scratch）、下のレイヤーを露出させる発掘体験。
*   **Hyper-local Evidence**: 「この辺り」ではない。「足元の半径数メートル」の事実だけを突きつける。何もなければ「Reaction: None (虚無)」と表示する潔さ。
