# ChronoHolidder (クロノホリダー) ⛏️🕰️

**「その場所、いつが一番盛り上がってた？」**

今立ってるその地面。10年前かもしれないし、江戸時代かもしれないし、なんなら人間がいなかったジュラ紀かもしれない。
嘘やまやかしのモックデータは一切なし。
WikidataとGBIFから「本当にそこにあった事実（Evidence）」だけを引っこ抜いて、その土地が一番輝いていた「ピーク」を可視化するアプリ。

## 思想 (Philosophy)
*   **No Mocks, All Real:** フェイクは絶対作らない。表示される画像もデータも、全部「本当にあった」資料だけ。
*   **Hyper-Resolution:** 「東京都」とかデカい括りじゃなくて、**「お前が今立ってるその足元」** の記憶を掘る。
*   **Seamless Time:** 「江戸」「明治」みたいな教科書の枠組みはどうでもいい。データが密集してるならそこがピーク。10年単位から1億年単位までシームレスに繋ぐ。

## 機能 (Features)
*   **Peak Era Extraction Engine**: 独自の密度解析アルゴリズムで、バラバラの史実データから「熱量の高い時代」を特定。
*   **Visual Evidence**: 口だけじゃなくて証拠を見せる。当時の写真、絵画、化石の標本写真を画面いっぱいに。
*   **Review Collection**: 掘り当てた時代は「コレクション」として保存。自分だけの歴史レイヤー持ち歩けるやつ。
