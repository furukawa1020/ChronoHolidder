# ChronoHolidder (クロノホリダー) ⛏️🕰️

**「ここ、昔なんだったの？」を、ガチで掘るアプリ。**

旅行先とか、近所の公園とかでふと思いません？
「100年前、ここ何があったんだろ？」
「いや、1億年前は恐竜いたんじゃね？」

ChronoHolidderは、そんな**「今立ってる場所のピーク（最高潮）」**を、地層ごとごっそり掘り起こしちゃうツールです。

---

## 思想 (きもち)

*   **見るな、掘れ (Dig, Don't Walk)**
    *   ただスマホ眺めるだけじゃつまんないですよね。画面の「現代の土」を指でガリガリ削ってください。その下から歴史が出てきます。
*   **全部いっしょ (Seamless Time)**
    *   「歴史」も「地質」も区別しません。去年のタピオカブームも、白亜紀のティラノサウルスも、どっちも大事な「過去」です。
*   **ここがピークだ (Thermal Peak)**
    *   何もない空き地でも、数万年前はマンモスのパーティー会場だったかもしれない。一番アツかった瞬間を探します。

---

## 中身はこんな感じ (実はけっこう凄い)

見た目はゆるいですが、裏側はガチガチに作り込みました。

### アプリ側 (Frontend)
*   **Flutter**: これで動いてます。
*   **AR Excavation**: カメラ越しに地面を削る機能。
*   **Riverpod**: 状態管理とかいう難しいやつ。

### サーバー側 (Backend)
*   **Go言語 (Golang)**:
    *   最初はPythonで作ってたんですが、「もっと速くしたい！」ってことで**Go**に書き換えました。爆速です。
*   **何してるの？**:
    *   Wikidata (歴史データ) と GBIF (化石データ) を混ぜて、独自の「盛り上がりスコア」を計算してます。
    *   「お城があったら +50点」「写真があったらボーナス」みたいな計算を、数億年規模でやってます。
*   **AI (Gemini)**:
    *   GoogleのAIが「つまりここはこういう場所だよ！」っていい感じにまとめてくれます。

---

## 遊び方

1.  **Backendを動かす**: `backend`フォルダで `go run main.go`
    *   (Docker派の人は `docker run` でもいけます)
2.  **Appを動かす**: `app`フォルダで `flutter run`
3.  **掘る**: Let's Dig!
